Here is a showcase of my best games. The games were created in a wide variety of programs, including Unity and Unreal Engine. I will add my role in the development of the game. If the game is available to download/play, I will provide a link.

Advoidment

This is a game that I created in Unity 2D in a team of 5 for our Casual Game Development class. The game has you completing bite sized minigames so you can get rid of them to press a bagel as many times as you can. I served as the main minigame programmer.

Funky Mungos

This is a game created in the Unreal Engine in a team of 7 for our IGME Production Studio class. The game is a short 3D collectathon where you must run around a level and catch all the Mungos that got loose. I worked as the Tech Art Lead, creating all the models and animations.

A downloadable version of the game will be available soon! You can check my work in the 'Models' section.

Don't Listen

This is a first person puzzle game created in Unity 3D in a team of 5 for our Game Design and Development II class. We were tasked to create a puzzle game set in the world of Ethshar, and this is our product after five weeks. I served as the Version Control Lead, a 3D Designer, and a Programmer.
